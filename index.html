How To Navigate Background Image Properties

Background image properties are primarily used in a Cascading Style Sheet (CSS) that is connected to your HTML file. When coding background images it is important to know how to edit the repeat, size, position, origin, and clip properties. By default, when embedding a background image, the image will repeat to cover the background area of your HTML element. This can sometimes cause your background to look crowded and make the content of your website harder to see and read if not edited properly. But when you utilize all the properties of background images it can add another dimension to your website and make it visually pleasing to users.

Although using a background image may seem overwhelming at first, it is a great tool to have under your belt when creating a website. Below you will find properties of background images broken into five categories: repeat, size, position, origin, and clip. In the five sections, there is a description of what each property has to offer and different ways to code them. After reviewing each section you will see that coding background images is not that bad when broken down. All it takes is some practice.

Repeat

The background-repeat function is automatically in use. In order to get rid of it you must put no-repeat if you do not want the background image to be repeated.

Some other background-repeat values you can use are:

  • repeat-x
  • repeat-y
  • space
  • round
  • space repeat
  • round space
Size

The background-size property can be defined by lengths (pixels — px), percentages (_%), or using the terms “contain” or “cover”. If you use the term contain for your background image it will size the background image to be as large as possible with respect to the ratio of the images’ original size. Which may result in blank spaces at the top and bottom or the left and right sides of the designated background area. Using the term cover will size the background image so that the area is completely covered by the image and most likely the image will overflow the area. Causing parts of the background image to not be visible in the designated background area.

Position

This property, background-position, sets the start position of the background image. The default of the background-position tends to be at the top-left corner and repeats vertically and horizontally. *Before coding your background-position, set your background-attachment value to fixed; as done in the example below.

Some other background-position values you can use are:

  • bottom right
  • 25% 50%
  • 50px 50px
Origin

The property, background-origin, pinpoints exactly where it is positioned. The values used are border-box, padding-box, and content-box. Border determines if the image starts at the upper left corner of the border, padding by default places the image at the upper left corner of the paddings edge, and content makes the background image start at the upper left corner of the content.

When using the value border you must specify the length (px), type (solid, dotted, dashed, etc.), and color.

For the padding, you can display the number of pixels (px).

Clip

The background-clip property refers to the painting area of the background. This property uses the same values as the background-origin property.

The key to mastering background image properties is to have fun and try out different variations until you find one you like. Don’t forget that you can also go back if you don’t like the outcome!

Image Hover Effects

The following section is dedicated to image hover effects, which are defined as visual changes that can take place when an operator's cursor is placed over an image; or tapped on when a website is accessed via a mobile device. Image hover effects, similar to background image properties, are a great way to spice up your website and make it appealing and interactive for users. Hover effects in general are a great way to help viewers keep track of where their cursor is. For example, when scrolling through a list it can be very helpful for the words to be highlighted when hovered over. Image hover effects create the ability for images to highlight important information, save space by having an image transform into another (rather than having to make space for multiple images), and so forth. Image hover effects can also simply be for aesthetic purposes.

When coding an image hover effect the place to do so is in a Cascading Style Sheet (CSS). Below will guide you through how to incorporate quick and easy hover effects to enhance your websites by sharing a few pointers, including example codes, for when embedding an image hover effect.
